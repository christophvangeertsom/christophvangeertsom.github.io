Dagboeken

2015/01/26 Dag 1

Als eerste kreeg ik een rondleiding door heel het gemeentehuis. Hierdoor heb ik dan kennis kunnen maken met al mijn collega’s wat er heel veel zijn. Daarna heb ik met Pieter (stagebegeleider) al een paar taken overlopen die ik komende twee weken moet maken. Op de computer die ik gebruik moest ik verschillende programma’s installeren waaronder Visual Studio om te kunnen programmeren. Mijn eerste taak was een programma dat Pieter geschreven had, iets uitbreiden. Het programma dat hij geschreven had laat verschillende foto’s zien op een tv-scherm. Hetgeen ik heb moeten doen is het programma zodanig uitbreiden dat er in een apart tekstbestand gegevens konden gezet worden en dat de code dus overzichtelijker zou worden.

2015/01/27 Dag 2

Het programma dat ik al een beetje had uitgebreid, heb ik nu nog meer uitgebreid. Ik heb het meeste van de dag op mijn bureau gewerkt aan dit project.

2015/01/28 Dag 3

In de voormiddag heb ik tijd gehad om aan mijn GIP-taken te werken omdat mijn stagebegeleider weg moest. Mijne eerste GIP-taak is afgeraakt, dit was de taak van de huisstijl. Hiervoor heb ik naar de dienst grafische vormgeving moeten wandelen om wat vragen te stellen. Ik ben ook begonnen aan de GIP-taak van bedrijfsbeheer. Om deze taak te kunnen maken heb ik naar verschillende diensten moeten gaan voor informatie. Deze diensten waren, financiële dienst, milieu, personeelsdienst.

2015/01/29 Dag 4

Dit was mijn eerste dag dat ik tot s’avonds acht uur zou moeten werken. Daarom ben ik er direct ingevlogen zodat de tijd sneller zou gaan. Ik ben begonnen met de taak van de huisstijl helemaal te maken in een markdown bestand dat zodra ik thuiskwam de taak kon uploaden op mijn website. De taak van bedrijfsbeheer over de analyse van het stagebedrijf heb ik ook voor ¾ in een markdown bestand gezet zodat ik deze ook zo snel mogelijk thuis kon uploaden. Nadat ik deze twee taken had omgezet in markdown bestanden, ben ik begonnen met het project dat ik gisteren had gekregen. Met dit project was het de bedoeling dat de mensen die bijvoorbeeld vacatures e.d. ingaven een gemeenschappelijk formulier hadden waar ze alleen maar tekstvakjes moesten invullen. Dit werd dan opgeslagen in een tekstbestand door op een knop te klikken. Door html-code werd het tekstbestand meteen opgeslagen in een html-page waardoor men dus altijd een standaard ‘template’ had. Dit project was al ietsje moeilijker dus ben ik maar direct aan de slag gegaan. Rond de middag had ik dit project af dus had ik het eigenlijk een beetje overschat. Het enige wat nu nog gedaan moest worden is de html-file juist linken aan dit formulier. Na de middagpauze ben ik dan weer aan de slag gegaan en een paar uurtjes later heb ik het kunnen linken. Toen ik vertelde dat ik klaar was heeft Pieter mij gezegt dat ik niet drie verschillende projecten aan het maken was, maar dat ik op het einde van mijn stage deze verschillende projecten moest samenbrengen tot één project. Wat dus uiteindelijk gebruikt gaat worden op het gemeentehuis. Nu ik dit weet ga ik jullie elke keer vertellen welke stap ik gemaakt heb en wat eigenlijk de bedoeling is. Later deze week zal ik ook uitleggen wat het eindresultaat zou moeten zijn van wat ik aan het maken ben. Na een lange maar zeer leerrijke dag ben ik thuisgekomen, heb ik mijn website geupdatet en ben ik in mijn bed gekropen. Klaar voor de halve dag die ik morgen heb.

2015/01/30 Dag 5

Na gisteren een lange dag te werken heb ik vandaag een halve dag verlof. Dit wil natuurlijk niet zeggen dat ik op die 4 uur dat ik moest werken niet hard heb gewerkt. Ik heb het eerste deel van mijn eind project afgewerkt. Dit was een formulier dat het onthaal kon gebruiken om te kiezen wellke vijf schermen ze op het tv scherm wilden hebben. Als ze dan op opslaan klikten werd de info naar een tekstbestand gestuurd. Het andere deel dat ik nu nog moet schrijven is wanneer de computer aan het tv scherm aangezet wordt, leest het direct de file in waar de info staat en laat het meteen de verschillende schermen zien. Dit is dus mijn eindproject. Hopelijk ga ik het halen. We zullen zien, nu ben ik snel verder gaan werken aan het nieuwe deel. Na een tijdje denken ben ik begonnen aan het laatste deel. Ik had gedacht dat het iets gemakkelijker ging want het was heel veel nadenken. Toch heb ik het niet opgegeven. Rond twaalf uur ben ik naar huis gegaan omdat ik een halve dag verlof kon nemen. Klaar voor een beetje rust.

2015/02/02 Dag 6

Na een ‘langer’ weekend was het de tweede week van de stage die van start ging. Ik ben dan maar direct aan de case study begonnen zodat wanneer Pieter aankwam ik hem een paar vragen kon stellen. Ook had Evelyne mij informatie doorgestuurd over de milieu voorwaarden van de taak van bedrijfsbeheer. Ik heb deze informatie dan direct aan mijn taak toegevoegd waardoor ze nu bijna af is. Nu ik de vragen gesteld heb kan ik beginnen aan de case study. Rond de middag was mijn taak van de case study klaar. Dan ben ik maar begonnen aan het project zodat dit zeker deze week klaar zou zijn. Nadat Pieter mij nog een beetj emeer uitleg heeft gegeven snapte ik al wat meer hoe ik de projecten moest samenbrengen. In de namiddag heb ik een vergadering bijgewoont in verband met een nieuw softwarepakket. Meer info hierover na de vergadering. De vergadering was eigenlijk een leverancier dat kwam praten over hun softwarepakket zodat de gemeente het zou gebruiken. Deze leverancier was DigiDoc, zij kwamen software voorstellen dat zo goed als alles digitaal doet, bijvoorbeeld zoals brieven, barcodes op producten zoals op laptops.
