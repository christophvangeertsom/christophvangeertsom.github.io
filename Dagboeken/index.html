Dagboeken

2015/01/26 Dag 1

Als eerste kreeg ik een rondleiding door heel het gemeentehuis. Hierdoor heb ik dan kennis kunnen maken met al mijn collega’s wat er heel veel zijn. Daarna heb ik met Pieter (stagebegeleider) al een paar taken overlopen die ik komende twee weken moet maken. Op de computer die ik gebruik moest ik verschillende programma’s installeren waaronder Visual Studio om te kunnen programmeren. Mijn eerste taak was een programma waarvan Pieter het begin van had geschreven, een groot deel uitbreiden. Het programma dat hij geschreven had laat verschillende foto’s zien op een tv-scherm. Hetgeen dat ik heb moeten doen is het programma zodanig uitbreiden dat er in een apart tekstbestand gegevens konden gezet worden, ook dat het scherm, wanneer de werkdag erop zit, uitvalt en wanneer de werkdag begint, het terug aanspringt. Het was ook de bedoeling dat ik object georiënteerd zou programmeren.

2015/01/27 Dag 2

Ik heb heel de dag aan dit project gewerkt en heb er 95% van kunnen maken. Alleen moest ik nog vinden hoe ik de schermen kon aan en uitzetten met C# code. Ik wist dat Pieter woensdagvoormiddag er niet ging zijn dus heb ik de GIP-taken nog even laten liggen.

2015/01/28 Dag 3

In de voormiddag heb ik tijd gehad om aan mijn GIP-taken te werken omdat mijn stagebegeleider weg moest in de voormiddag. Mijne eerste GIP-taak is afgeraakt, dit was de taak van de huisstijl. Hiervoor heb ik naar de dienst grafische vormgeving moeten wandelen om wat vragen te stellen. Ik ben ook begonnen aan de GIP-taak van bedrijfsbeheer. Om deze taak te kunnen maken heb ik naar verschillende diensten moeten wandelen voor informatie. Deze diensten waren, financiële dienst, milieu, personeelsdienst. In de namiddag heb ik dan eindelijk mijn eertse project kunnen afmaken. Dit zal ook gebruikt worden door de gemeente waardoor ik dus trots kan zijn op mijn project. Daarna heb ik een nieuwe opdracht gekregen. Deze is wel iets moeilijker maar ik ga proberen om dit tot een goed einde te brengen. Over dit project morgen meer uitleg.
